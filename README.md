
# QuickTools

QuickTools is a .NET web application which features various useful tools and converters.

List of Tools:
* Calculator
* Color Picker
* Stopwatch
* Timer
* Notes
* Calendar
* Graph Builder

Converters:
* Temperature
* Volume
* Length
* Mass
